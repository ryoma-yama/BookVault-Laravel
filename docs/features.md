# 機能一覧

BookVault-Laravelが提供する主要機能について説明します。

## 📖 書籍管理機能

### 一般ユーザー向け機能

#### 書籍の閲覧
- 書籍一覧の表示
- 書籍詳細情報の表示
- 著者情報の表示
- タグによる分類表示
- レビューと評価の閲覧

#### 書籍検索
- キーワード検索
- 著者名検索
- タグによる絞り込み
- ISBN検索
- Meilisearchによる高速全文検索

### 管理者向け機能

#### 書籍の管理
- 書籍の新規登録
- 書籍情報の編集
- 書籍の削除
- Google Books API連携によるISBN検索と自動入力
- 複数の著者の関連付け
- タグの設定と管理

#### 在庫管理（BookCopy）
- 蔵書コピーの登録
- 在庫状態の管理（利用可能/貸出中/修理中など）
- 所蔵場所の記録
- 個別識別番号の管理

## 👥 ユーザー管理機能

### 認証機能（Laravel Fortify）

#### 基本認証
- ユーザー登録
- ログイン / ログアウト
- パスワードリセット
- メールアドレス確認

#### セキュリティ
- 二要素認証（2FA/TOTP）
- パスワード確認
- セッション管理

### ユーザープロフィール管理

#### プロフィール設定
- 表示名の変更
- メールアドレスの変更
- パスワードの変更
- アカウントの削除

#### 外観設定
- テーマの切り替え（ライト/ダーク）
- 個人設定の保存

### 管理者機能

#### ユーザー管理
- ユーザー一覧の表示
- ユーザー役割（ロール）の変更
  - 一般ユーザー（user）
  - 管理者（admin）
- ユーザー情報の更新

## 📚 貸出・予約管理機能

### 貸出機能

#### 貸出処理
- 書籍の貸出申請
- 貸出状態の追跡
- 返却処理
- 延滞管理

#### 貸出履歴
- ユーザーごとの貸出履歴
- 現在の貸出状況確認
- 返却予定日の管理

### 予約機能

#### 予約処理
- 貸出中書籍の予約
- 予約状態の管理
- 予約キャンセル
- 予約順位の管理

#### 予約通知
- 予約可能通知
- 予約期限管理

## ⭐ レビュー・評価機能

### レビュー投稿
- 書籍へのレビュー投稿
- 星評価（5段階）
- レビューの編集
- レビューの削除

### レビュー閲覧
- 書籍ごとのレビュー一覧
- 平均評価の表示
- レビュー投稿者情報の表示

## 🔍 検索・フィルタリング機能

### 高度な検索
- Meilisearchを使用した全文検索
- リアルタイム検索サジェスト
- 複数条件での絞り込み
- 検索結果のハイライト表示

### フィルタリング
- カテゴリー別フィルター
- タグによるフィルター
- 著者によるフィルター
- 在庫状況によるフィルター

## 🎨 UI/UX機能

### レスポンシブデザイン
- モバイル対応
- タブレット対応
- デスクトップ対応

### アクセシビリティ
- キーボードナビゲーション対応
- スクリーンリーダー対応
- ARIAラベルの実装

### ダークモード
- システム設定の自動検出
- 手動切り替え
- ユーザー設定の保存

## 🔐 セキュリティ機能

### 認証・認可
- ミドルウェアによるアクセス制御
- ロールベースの権限管理
- CSRF保護
- XSS対策

### データ保護
- パスワードのハッシュ化（bcrypt）
- セキュアなセッション管理
- HTTPSの推奨

## 📊 管理者ダッシュボード

### 統計情報
- 総書籍数
- 総ユーザー数
- 現在の貸出数
- システム稼働状況

### クイックアクション
- 新規書籍登録
- ユーザー管理
- 貸出管理
- システム設定

## 🔧 API機能

### 外部API連携
- Google Books API連携
- ISBN検索による書籍情報自動取得
- 書影の自動取得

### RESTful API
- Loan API（貸出管理）
- Reservation API（予約管理）
- 認証済みリクエストのサポート
